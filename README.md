# 🛠️ DCS Translation TOOL

**A tool for text translation of missions and campaigns in DCS World**

![DCS Translation Tool Interface](https://github.com/Hokum-A/dcs-translation-tool/blob/main/assets/TOOL_1.2en.jpg)
*Main interface of the translation tool*

Supports `.lua` (dictionary), `.cmp` (campaign) files and mission `.miz` archives. The interface is fully translated into English and Russian.

> The program was created using generative AI and is being refined manually. Any help and constructive criticism is welcome!

---

**For Russian users:** [Русская версия описания доступна ниже](#russian-version)

---

## ✨ Features

*   **DCS File Parsing**: Reading and processing Lua `dictionary` files
*   **Mission Archive Handling**: Safe extraction and replacement of localization files inside `.miz` archives
*   **Locale Management**: Convenient work with localization folders directly from the interface
*   **Intelligent Filters**: Smart filtering of keys (`ActionText`, `description`, etc.) for translation
*   **Format Preservation**: Integrity control of escape sequences and line breaks
*   **Bilingual Interface**: Full support for English and Russian languages
*   **Dark Theme**: Interface optimized for comfortable work
*   **Context Management**: Create, edit, and save context for the translator
*   **Built-in Help**: Detailed guide for quick start
*   **No API Keys Required**: Use of publicly available services (DeepSeek, ChatGPT, Google Translate, etc.)
*   **Quick Translation**: All mission text in one clipboard for translation in just a few clicks

## 🚀 Installation and Launch

1. Go to **[Releases](https://github.com/Hokum-A/dcs-translation-tool/releases)**
2. Download the latest version of `DCS_Translation_TOOL_vX.X.exe`
3. Run the downloaded file

# 🛠️ DCS Translation TOOL

**Инструмент для текстового перевода миссий и кампаний игры DCS World**

![Интерфейс DCS Translation Tool](https://github.com/Hokum-A/dcs-translation-tool/blob/main/assets/TOOL_1.2ru.jpg)
*Главное окно программы для перевода*

Поддерживает работу с файлами `.lua` (dictionary), `.cmp` (файлы кампаний) и архивами миссий `.miz`. Интерфейс полностью переведён на русский и английский языки.

> Программа создана с использованием генеративного ИИ и дорабатывается вручную. Любая помощь и конструктивная критика приветствуется!

## ✨ Возможности

*   **Парсинг файлов DCS**: Чтение и обработка Lua-файлов `dictionary`
*   **Работа с архивами миссий**: Безопасное извлечение и замена файлов локализации внутри `.miz` архивов
*   **Управление локалями**: Удобная работа с папками локализации прямо из интерфейса
*   **Интеллектуальные фильтры**: Умная фильтрация ключей (`ActionText`, `description` и др.) для перевода
*   **Сохранение форматирования**: Контроль целостности escape-последовательностей и переносов строк
*   **Двуязычный интерфейс**: Полная поддержка русского и английского языков
*   **Тёмная тема**: Оптимизированный для работы интерфейс
*   **Управление контекстом**: Создание, редактирование и сохранение контекста для переводчика
*   **Встроенная справка**: Подробное руководство для быстрого старта
*   **Никаких API ключей**: Использование общедоступных сервисов (DeepSeek, ChatGPT, Google Translate и др.)
*   **Быстрый перевод**: Весь текст миссии в одном буфере обмена для перевода в несколько кликов

## 🚀 Установка и запуск

2. Скачайте последнюю версию `DCS_Translation_TOOL_vX.X.exe`
3. Запустите скачанный файл
