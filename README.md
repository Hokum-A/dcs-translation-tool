## ‚ú® Features

*   **DCS File Parsing**: Reading and processing Lua `dictionary` files
*   **Mission Archive Handling**: Safe extraction and replacement of localization files inside `.miz` archives
*   **Locale Management**: Convenient work with localization folders directly from the interface
*   **Intelligent Filters**: Smart filtering of keys (`ActionText`, `description`, etc.) for translation
*   **Format Preservation**: Integrity control of escape sequences and line breaks
*   **Bilingual Interface**: Full support for English and Russian languages
*   **Dark Theme**: Interface optimized for comfortable work
*   **Context Management**: Create, edit, and save context for the translator
*   **Built-in Help**: Detailed guide for quick start
*   **No API Keys Required**: Use of publicly available services (DeepSeek, ChatGPT, Google Translate, etc.)
*   **Quick Translation**: All mission text in one clipboard for translation in just a few clicks
